# Steganography
Created for CyberStorm 2019, Python 2.7 Optimized for Linux Cmd, Steganography Encryption and Decryption
# How to
command: python steg.py -(bB) -(sr) -o<val> [-i<val>] -w<val> [-h<val>]
(notes: -b = bit,
       -B = byte,
       -s = store data,
       -r = retrieve data,
       -o<val> = set offset,
       -i<val> = set interval,
       -w<val> = set wrapper file,
       -h<val> = set hidden file)
